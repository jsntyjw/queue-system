# WOG Queue System

## Description

To manage crowd control that allows walk-in customers or customers with prior-appointment made to get a queue for their turn to be “served”. 

## Installation

Files to run in terminal:

1. Frontend folder: 
    ```
        cd frontend
        npm start
    ```


    b. backend folder: 
        1. cd backend
        2. npm start
    c. QueueAPI folder:
        1. cd QueueAPI
        2. npm start
